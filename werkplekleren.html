Overzicht van opdrachten en PE’s

Week 2 - LockDownBrowser (evaluatie)

Beschrijving: Evaluatie van kennis en vaardigheden via LockDownBrowser.

Persoonlijke uitdagingen: N.V.T

Wat ik heb bijgeleerd: Goed op voorhand zien dat je doet wat er van je gevraagd wordt ivm de installatie van de LockDownBrowser.

Week 3 - Scratch project (opdracht)

Beschrijving: Ontwikkelen van een interactieve applicatie in Scratch.

Persoonlijke uitdagingen: Beginnen vanaf 0 en proberen een zo creatief mogelijk design te maken.

Wat ik heb bijgeleerd: Basisprincipes van programmeren op een relatief makkelijk niveau.

Week 4 - Git & GitHub (evaluatie) / Reflectie Scratch project (opdracht)

Beschrijving: Evaluatie van Git-vaardigheden en reflectie op eerder Scratch project.

Persoonlijke uitdagingen: Correct gebruik van Git-commando’s en kritisch reflecteren op eigen werk.

Wat ik heb bijgeleerd: Hoe dat Git werkt en dat zelfreflectie nuttig is om van je fouten te leren.

Week 5 - Gastsprekers (evaluatie) / Planning (opdracht)

Beschrijving: Evaluatie van gastsprekers en het opstellen van een persoonlijke planning.

Persoonlijke uitdagingen: Tijd effectief indelen en me hier ook aan houden.

Wat ik heb bijgeleerd: Planningsvaardigheden verbeteren en een beter zicht hoe het er aan toe gaat in de IT wereld.

Week 7 - C# Console (evaluatie) / POP Carrièrekompas (opdracht)

Beschrijving: Evaluatie van C#-vaardigheden en invullen van POP Carrièrekompas.

Persoonlijke uitdagingen: De klik maken met klassen in C# en kritisch nadenken over mijn carrièrekeuzes.

Wat ik heb bijgeleerd: Meer oefenen met klassen in C# en inzicht in persoonlijke sterktes/zwaktes.

Week 8 - Feedback evaluatie C# Console (opdracht)

Beschrijving: Verwerken van feedback op het eerder gemaakte C# project.

Persoonlijke uitdagingen: Feedback correct interpreteren en toepassen in de code.

Wat ik heb bijgeleerd: Hoe ik mijn code kan verbeteren en optimaliseren op basis van feedback.

Week 9 - HTML & CSS (evaluatie)

Beschrijving: Evaluatie van kennis en vaardigheden in HTML en CSS.

Persoonlijke uitdagingen: Tijdsdruk en structuur opbouwen.

Wat ik heb bijgeleerd: Dat ik niet moet blijven hangen bij een bepaald aspect aangezien ik dan in tijdsnood kom voor de rest.

Week 10 - Debugging (evaluatie) / Feedback evaluatie debugging (opdracht)

Beschrijving: Problemen oplossen in bestaande code en feedback verwerken.

Persoonlijke uitdagingen: Problemen identificeren en systematisch oplossen.

Wat ik heb bijgeleerd: Debugging-technieken toepassen en mijn aanpak voor de PE's verbeteren.

Week 11 - SDG's (evaluatie)

Beschrijving: Evaluatie van kennis rond Sustainable Development Goals.

Persoonlijke uitdagingen: N.V.T

Wat ik heb bijgeleerd: Ik heb vooral veel bijgeleerd van de presentaties van de gastsprekers.

Week 12 - Curriculum Vitae (opdracht)

Beschrijving: Opstellen van een professioneel CV.

Persoonlijke uitdagingen: De code zo efficiënt mogelijk schrijven rekening houdende met de media queries.

Wat ik heb bijgeleerd: Dat de laatste 20% van de opdracht vaak het langst duur.

Week 13 - C#WPF (evaluatie) / Portfolio (opdracht)

Beschrijving: Evaluatie van kennis betreft C# WPF en mijn eigen portfolio maken

Persoonlijke uitdagingen: De code behind achter de XAML code en opnieuw reflecteren over de voorbije maanden en dit omzetten tot een portfolio.

Wat ik heb bijgeleerd: Dat de code behind nog steeds een werkpunt is en dat reflecteren over mezelf vaak niet zo simpel is als het lijkt.
